🚀 Chạy Dashboard trên máy cục bộ

Dashboard Streamlit yêu cầu Python backend. Thực hiện theo các bước sau:

1
Clone repository
git clone https://github.com/wane-bs/phantichtcdn.git
cd phantichtcdn
2
Cài đặt thư viện
pip install -r requirements.txt
Cần: Python 3.9+, pandas, plotly, streamlit, scikit-learn, openpyxl, statsmodels
3
Khởi chạy Dashboard
streamlit run src/app.py
Mở trình duyệt tại http://localhost:8501 — Nhấn nút "🚀 Chạy Pipeline" trên sidebar để cập nhật dữ liệu.
